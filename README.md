<h1>Dev finances</h1>

![Dev finances](https://github.com/VitorHugoAntunes/Maratona_Discover/blob/master/assets/presentation_devFinances_.JPG "Dev finances application")

Application created using HTML, CSS and JS. Its purpose is to calculate and control transactions and add them to a table that stores the data in LocalStorage.
The project was created during a programming marathon made available by [Rocketseat](https://github.com/Rocketseat).
